# kubetoolbox

This is a troubleshooting container by Alpine, GoTTY and a lot of troubleshooting tools.

### Installed tools
#### Webshell
- GoTTY

#### Network
- busybox-extras
- libc6-compat
- bind-tools
- curl
- iproute2
- tcpdump
- iperf
- iftop
- openssh-client

#### Kubernetes
- kubectl

#### Others
- jq
